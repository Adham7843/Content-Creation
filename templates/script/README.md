# Script Templates

Frameworks for spoken-word content: videos, podcasts, webinars, and live streams.

## Script Types

| Template | Medium | Length |
|----------|--------|--------|
| `youtube-script.md` | YouTube | 3-15 min |
| `podcast-script.md` | Podcast | 15-60 min |
| `webinar-script.md` | Webinar | 30-60 min |
| `live-stream.md` | Live (Twitch/Youtube/LinkedIn) | 30-120 min |
| `voiceover.md` | Video voiceovers | 30s-5min |
| `short-form.md` | TikTok/Reels/Shorts | 15-60s |

## Script Structure

```markdown
---
type: [youtube/podcast/webinar/live/voiceover/short]
duration: [target length]
tone: [professional/casual/educational/entertaining]
audience: [target viewer]
---

## Hook (0-30s)
[Opening — must stop the scroll / grab attention]

## Intro (30-60s)
[Who you are, what this is about, why it matters]

## Body (varies)
### Section 1: [Topic]
[Key points, stories, data, examples]

### Section 2: [Topic]
[Continue with structure]

### Section 3: [Topic]
[Wrap up main content]

## CTA / Outro
[What should they do next? Like, subscribe, comment, click link]

## Timestamps (for long-form)
0:00 - Hook
0:30 - Intro
2:00 - Main Content
...
```

## Tips

- **Read aloud:** Always test scripts by reading aloud
- **Conversational:** Write like you speak, not like you write
- **Pace:** 150 words per minute is average speaking pace
- **Transitions:** Use "Now," "But here's the thing," "Speaking of which"
- **CTAs:** Place early AND late — some viewers won't make it to the end
