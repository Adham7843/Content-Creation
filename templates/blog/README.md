# Blog Templates

Frameworks for blog posts optimized for SEO, readability, and conversions.

## Templates Available

| Template | Use Case | Structure |
|----------|----------|-----------|
| `how-to-guide.md` | Step-by-step tutorials | Intro → Prerequisites → Steps → Summary |
| `listicle.md` | Curated lists (Top N) | Intro → N items with details → Conclusion |
| `thought-leadership.md` | Industry opinions/insights | Hook → Problem → Perspective → Call to action |
| `case-study.md` | Customer success stories | Background → Challenge → Solution → Results |
| `comparison.md` | X vs Y comparisons | Overview → Features table → Winner → Verdict |
| `news-announcement.md` | Product/news announcements | What → Why → Impact → Next steps |
| `ultimate-guide.md` | Comprehensive deep-dives | Table of contents → Sections → FAQ |

## SEO Best Practices

- **Title:** Include primary keyword, keep under 60 chars
- **H1:** Use the title
- **H2/H3:** Include secondary keywords naturally
- **Meta description:** 150-160 chars with keyword
- **URL:** Short, keyword-rich, lowercase with hyphens
- **Images:** Alt text with keywords
- **Internal links:** Link to 2-3 related posts
- **External links:** Link to 1-2 authority sources

## Blog Post Frontmatter

```yaml
---
title: "Post Title"
date: YYYY-MM-DD
author: Author Name
tags: [keyword1, keyword2, keyword3]
description: "SEO meta description 150-160 chars"
canonical: https://example.com/blog/post
image: /images/blog/post-hero.jpg
draft: true
---
```
