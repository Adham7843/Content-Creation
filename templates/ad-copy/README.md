# Ad Copy Templates

Templates for paid advertising across platforms.

## Platforms

| Template | Platform | Format |
|----------|----------|--------|
| `facebook-ad.md` | Facebook/Instagram | Single image, carousel, video |
| `google-search.md` | Google Search | Text ad with headlines + descriptions |
| `google-display.md` | Google Display Network | Image + headline + description |
| `linkedin-ad.md` | LinkedIn | Sponsored content, InMail, text ads |
| `twitter-ad.md` | Twitter/X | Promoted tweets |
| `tiktok-ad.md` | TikTok | In-feed video ads |
| `native-ad.md` | Taboola/Outbrain | Sponsored content units |

## Structure

```markdown
---
platform: [facebook/google/linkedin/twitter/tiktok/native]
format: [image/video/carousel/text]
objective: [awareness/consideration/conversion]
audience: [target demographic]
---

## Headline
[Grab attention, include keywords]

## Primary Text / Body
[Main copy — problem, solution, social proof]

## CTA
[Button text — be specific]

## Visual Description
[What the image/video shows]

## Landing Page
[Where they go — must match ad promise]
```

## Copywriting Principles

- **AIDA:** Attention → Interest → Desire → Action
- **Features vs Benefits:** Sell the benefit, not the feature
- **Social Proof:** "Join 10,000+ happy customers"
- **Scarcity:** "Limited time offer", "Only 5 spots left"
- **Specificity:** "Get 3x more leads" vs "Get more leads"
