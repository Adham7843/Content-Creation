# Social Media Templates

Platform-specific post templates for consistent, high-engagement content.

## Platforms

| Platform | Templates | Best For |
|----------|-----------|----------|
| **Twitter / X** | threads/ | Short announcements, threads, engagement posts |
| **LinkedIn** | posts/ | Professional content, thought leadership, company updates |
| **Instagram** | posts/ | Visual posts, carousels, Stories |
| **TikTok** | scripts/ | Short-form video scripts |
| **Facebook** | posts/ | Community engagement, longer posts |

## Post Structure

Every post template follows this format:

```markdown
---
platform: [twitter/linkedin/instagram/tiktok/facebook]
type: [post/thread/carousel/story]
tone: [professional/casual/inspirational/humorous]
target: [B2B/B2C/mixed]
---

## Headline / Hook
[First line — must grab attention]

## Body
[2-3 paragraphs or bullet points]

## CTA
[What action should they take?]

## Hashtags
#relevant #tags #here

## Media
[Image/Video description if applicable]
```

## Engagement Tips

- **Twitter/X:** First 280 chars are critical. Use threads for depth.
- **LinkedIn:** Lead with a personal story or strong opinion. 900-1200 words performs best.
- **Instagram:** Carousels get highest engagement. First image is everything.
- **TikTok:** Hook in first 2 seconds. Use trending sounds.
- **Facebook:** Longer posts work. Ask questions to boost comments.
